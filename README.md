
# .NET MAUI template

Application developed in .NET MAUI version .NET 7 (iOS and Android) 


## Features

- MVVM architecture
- Custom controllers (button, input, tabbar ...)
- Session managements 
- CommunityToolkit
- Restsharp
- Newtonsoft Json
- Fingerprint
